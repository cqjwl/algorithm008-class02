学习笔记

1. 数组、链表、跳表、栈、队列的时间复杂度

|      | 数组 | 链表 | 跳表    | 栈   | 队列 | 优先队列 |
| ---- | ---- | ---- | ------- | ---- | ---- | -------- |
| 查询 | O(n) | O(n) | O(logn) | O(n) | O(n) | O(n)     |
| 插入 | O(n) | O(1) | O(logn) | O(1) | O(1) | O(1)     |
| 删除 | O(n) | O(1) | O(logn) | O(1) | O(1) | O(logn)  |

2. 链表：单向链表、双向链表、循环链表；
3. 跳表：有序链表，升维，空间换时间；
4. 栈：先入后出；
5. 队列：先入后出；
